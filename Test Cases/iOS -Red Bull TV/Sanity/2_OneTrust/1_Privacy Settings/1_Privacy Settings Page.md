Precondition
```
# A new instance of `<application>` is installed
1. Launch `<application>` from a fresh install
2. Observe the Consent Management Prompt (CMP)
3. Select the "Privacy Settings" Button
```

Step 1:
```
Verify the page Title
```
```
The page Title is displayed at the top of the page
- Text - "Privacy Settings"
- Alignment - Center

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_privacysettings_title.png)
```

Step 2:
```
Verify the page Copy
```
```
The Copy is displayed below the Title.
- Text - Clearly states the purpose of the SDKs
- Alignment - Center

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_privacysettings_copy.png)
```

Step 3:
```
Verify each Copy link is selectable
```
```
Selecting a Copy link opens the associated web page

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_privacysettings_copylinks.png)
```
