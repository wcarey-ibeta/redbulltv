Precondition
```
# A new instance of `<application>` is installed
1. Launch `<application>` from a fresh install
2. Observe the Consent Management Prompt (CMP)
3. Select the "Privacy Settings" Button
```

Step 1:
```
Verify the Performance SDK Option
```
```
The Performance SDK option has:
- Text - Performance
- Toggle - Default Position "OFF"

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_privacysettings_performance.png)
```

Step 2:
```
Select the "Performance" option label
```
```
The user should be navigated to the Performance page

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_performance_page.jpeg)
```

Step 3:
```
Verify the "Performance" Page Toggle
```
```
The "Performance" option should display at the top of the page.
- Text - Performance
- Toggle - should reflect the state of the toggle on the main Privacy Settings page

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_performance_page_toggle.png)
```

Step 4:
```
Verify the page copy
```
```
The page Copy will display below the "Performance" Option
Alignment - Left aligned

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_performance_page_copy.png)
```

Step 5:
```
Select the View SDK List
```
```
The user is redirected to the SDK List page and only performance SDKs should appear

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_performance_page_sdkbutton.png)
```

Step 6:
```
Verify the Back button
```
```
The back button displays at the top left of the page
- Icon - Left Chevron (White)
- Selection - User is returned to the previous Page

![](https://wcarey-ibeta.github.io/redbulltv/images/ios/onetrust/rbtv_performance_page_back.png)
```
